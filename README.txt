README

Open the code in the NetLogo application or website (the application works better).

Go to the interface tab and press the setup button and then the go button.

To reset, press the startup button again, and to stop, press the go button again

Sliders can be adjusted while running to see how the behaviour changes with different parameters.

You may notice that the boids do occasionally go over the obstacles which is due to an approximation of angles and the fact that there is nothing that can specifically block the boids from being on the obstacles. However, for the most part you will see that the boids go around the obstacles and do not "collide" with them.

Citations for base model:

Wilensky, U. (1998). NetLogo Flocking model. http://ccl.northwestern.edu/netlogo/models/Flocking. Center for Connected Learning and Computer-Based Modeling, Northwestern University, Evanston, IL.
